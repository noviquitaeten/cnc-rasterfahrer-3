Cnc-rasterfahrer-3

convert pixel bitmap image to g-code for cnc-mills

Download this project as a .zip file Download this project as a tar.gz file

you have got a desktop cnc mill?

then take your last weekend trip photos and cnc-drill them to art or scratch a scanned treasure map into rusty scrap metal. The rasterfahrer can convert most bitmap images to G-code. G-code is the cnc-mill mashine code.

how to use

You will need the processing SDK from processing.org to compile the source code. Then download the zip or tar file. Copy the rasterfahrer and the library folder into your processing sketchbook folder. Open the rasterfahrer in processing and run it. Now load your image, slide the sliders, hit the buttons and export your ready to use G-code.

features

drill dot patterns line patterns edge detection patterns noise, to look less computer generated
